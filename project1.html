Car Parts

● A Single Page Application (SPA) for buy car parts in bulk.
● The data on the website is stored in MongoDB and fetched using a server-side API, which I implemented using Express JS.
● Firebase Authentication (Google Sign-in & Email/Password Login) was implemented. For security reasons, the Private Route and Admin Route have been implemented.
● Users and administrators each have their own dashboard with distinct features.
● User can order car parts.
● User can pay for this order.
● User can add reviews.
● Admin can delete unpaid order.
● Admin can change status of an paid order from pending to shipped.
● Admin can make any user admin.
